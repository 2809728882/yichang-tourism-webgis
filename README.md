# 宜游智图 WebGIS

面向宜昌游客的 WebGIS + AI 智能旅游系统（移动端优先）。

## 当前能力

- 地图主视图：POI、客流热度、路线编号点、真实道路线路绘制
- AI 路线规划：支持天数、时长、偏好、交通模式、人数
- AI 智能问答：普通问答 + 客流问答 + 实时天气问答
 -  示例
- 你：带老人适合去哪？
助手：行程概览
- 主题：轻松慢游，适合老人，减少步行压力  
- 交通：以打车为主，结合短途观光车，避免长时间步行  

第一天：城区休闲游  
1. 上午（3小时）  
   - 交通：从丽橙酒店打车至 滨江公园（约10分钟）  
   - 活动：沿江散步观景，可坐长椅休息，欣赏三峡风光  
   - 贴士：公园步道平坦，有树荫，适合老人慢走  

2. 中午（1.5小时）  
   - 午餐：打车至 陶珠路美食街（约8分钟），推荐清淡菜系如“三七砂锅”  

3. 下午（3小时）  
   - 交通：打车至 宜昌博物馆（约15分钟）  
   - 活动：室内参观，了解三峡文化，空调充足可避暑  
   - 贴士：博物馆有休息区，可租轮椅（需提前电话确认）  

4. 傍晚（2小时）  
   - 交通：打车回酒店休息后，步行至 CBD购物中心（距酒店约500米）  
   - 活动：逛商场或周边小吃街，晚餐可选“自己人酒楼”  

第二天：三峡大坝轻体验  
1. 上午（4小时）  
   - 交通：提前预约 包车/专车（往返约200元，酒店可协助）  
   - 活动：直达 三峡大坝坛子岭观景台，乘坐电瓶车（30元/人）游览  
   - 贴士：避开团队高峰，专车可停靠近入口，减少步行  

2. 中午（1.5小时）  
   - 午餐：景区附近餐厅“三峡工程酒店”用餐，菜品清淡  

3. 下午（3小时）  
   - 活动：乘车至 截流纪念园，平坦步道观赏大坝全景  
   - 返程：专车送回酒店休息  

第三天：自然风光轻徒步  
1. 上午（3小时）  
   - 交通：打车至 三游洞景区（约20分钟）  
   - 活动：乘坐景区观光车游览，重点看 西陵峡口、摩崖石刻  
   - 贴士：部分路段有台阶，可选择平坦区域游览  

2. 中午（2小时）  
   - 午餐：景区出口“放翁酒家”用餐，临江观景  

3. 下午（2小时）  
   - 返程：打车回酒店取行李，结束行程  

交通与贴士  
1. 打车优先：宜昌打车方便，起步价8元，景点间通常不超过30元  
2. 观光车利用：大坝、三游洞等景区内尽量选电瓶车，节省体力  
3. 休息安排：每2-3小时安排一次休息，中午回酒店小憩  
4. 应急准备：随身携带常用药、温水，避开正午高温时段  

可根据老人体力灵活调整，如第三天较累可改为城区游览（夷陵广场、儿童公园）

- 移动端适配：左侧功能面板 + 地图主视图 + 悬浮信息卡

## 技术栈

- 前端：React + Vite + Leaflet
- 后端：Node.js + Express
- 外部服务：高德 API（天气、路况、路径）、Open-Meteo（免费天气备选）
- AI：OpenAI 兼容接口（可配置 DeepSeek）

## 目录结构

- `frontend`：前端工程
- `backend`：后端 API
- `docker-compose.yml`：容器化启动模板

## 本地启动

### 1) 启动后端

```bash
cd backend
npm install
copy .env.example .env
npm run dev
```

默认端口：`3001`

### 2) 启动前端

```bash
cd frontend
npm install
npm run dev
```

默认端口：`888`

访问地址：`http://127.0.0.1:888/`

## 主要 API

- `GET /api/health`
- `GET /api/pois`
- `GET /api/realtime`
- `POST /api/plan-route`
- `POST /api/chat`
- `POST /api/route/geometry`（真实道路几何）
- `GET /api/user/:userId/itineraries`
- `POST /api/user/:userId/itineraries`
- `DELETE /api/user/:userId/itineraries/:id`
- `POST /api/feedback`

## 说明

- 若高德路况接口异常，系统会自动降级并给出可读提示。
- 若 AI 服务不可用，路线规划会自动回退到启发式方案。
- 生产环境请确保 `.env` 不入库，避免泄露密钥。
